# Requirements

## Backend (Python)

```bash
cd backend
pip install -r requirements.txt
```

Packages:
- fastapi
- uvicorn
- boto3
- python-dotenv
- pydantic

## Frontend (Node.js)

```bash
cd frontend
npm install
```

Packages:
- next
- react
- react-dom
- framer-motion
- axios
- tailwindcss
- autoprefixer
- postcss

## Run

Terminal 1:
```bash
cd backend
python main.py
```

Terminal 2:
```bash
cd frontend
npm run dev
```
